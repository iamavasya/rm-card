# ProjectK

**ProjectK** — це внутрішня ERP-система (Enterprise Resource Planning), розроблена для автоматизації менеджменту та цифровізації процесів у організації Пласт. Проєкт фокусується на централізації даних, які зазвичай розрізнені між паперовими архівами та таблицями.

## 🚀 Проблема та Рішення

У великих волонтерських спільнотах управління кадрами, фінансами та прогресом учасників часто відбувається неефективно через відсутність єдиної точки правди. 

**Project K** вирішує це шляхом створення єдиної екосистеми, де кожен рівень організації (від керівників до рядових учасників) має доступ до необхідних інструментів аналітики та управління в реальному часі.

## 🛠 Ключовий функціонал

Система розділена на кілька логічних модулів:

* **Personnel Management (CRM):** Ієрархічна структура користувачів (адміністратори, ментори, учасники). Ведення реєстрів малих груп та персональних профілів.
* **Achievement Engine:** Система відстеження прогресу (gamification). Візуалізація здачі іспитів, отримання сертифікатів та внутрішніх досягнень учасників.
* **Financial Ledger:** Модуль для моніторингу членських внесків. Дозволяє бачити відсоток оплати по підрозділах та виявляти заборгованості.
* **Activity & Event Logging:** Архівування історії заходів, таборів та важливих дат організації з можливістю аналізу активності за минулі періоди.
* **Gamification (Scoring System):** Система внутрішнього рейтингу ("точкування"), яка дозволяє порівнювати активність учасників на основі заданих алгоритмів.

## Технічний стек

Проєкт розробляється з акцентом на надійність та довговічність даних:

-   **Core:** .NET 10 / C#
-   **Database:** SQL Server + EF Core
-   **Architecture:** Modular Monolith
-   **Frontend:** Angular

## 🎯 Ціль проєкту

Створити стабільну платформу, яка перетворює хаотичні записи організації на структуровану базу даних, забезпечуючи цифрову тяглість та спрощуючи життя адміністраторам через автоматизацію рутинних завдань.
